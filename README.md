# coding_challenges_ccwc

https://codingchallenges.fyi/challenges/challenge-wc/

An implementation of the coding challenges wc exercise, written in golang.

## Quickstart:

```sh
go build
go install

ccwc test.txt
ccwc -c test.txt
ccwc -l test.txt
ccwc -w test.txt
ccwc -m test.txt

ccwc test.txt

cat test.txt | ccwc
```
